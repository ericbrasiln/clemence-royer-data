# clemence-royer-data

Dados e visualização inicial dos dados recolhidos com a ferramenta pyHDB relativos à Clémence Royer para pesquisa de doutorado de Yaci Farias (UFBA).

O tratamento dos dados foi realizado com os scripts desenvolvidos por Eric Brasil (UNILAB) disponível em https://github.com/ericbrasiln/analise_pyHDB. Esses scripts geram visualizações gráficas a partir dos dados coletados com a ferramenta pyHDB[^pyhdb] e compõe o artigo "pyHDB - Ferramenta Heurística para a Hemeroteca Digital Brasileira: utilizando técnicas de web scraping para a pesquisa em História"[^artigo-pyhdb]. 

[^pyhdb]: Brasil, E. (2021). pyHDB: Ferramenta heurística para a Hemeroteca Digital Brasileira (Version 1.5) [Computer software]. https://doi.org/10.5281/zenodo.5696670

[^artigo-pyhdb]: Brasil, E. (2021). pyHDB - Ferramenta Heurística para a Hemeroteca Digital Brasileira: utilizando técnicas de web scraping para a pesquisa em História. Revista História da historiografia, 2022 (no prêlo).
